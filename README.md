# Análise da saúde mental de estudantes

Projeto 1 da trilha Analista de dados associado em SQL da plataforma de cursos DataCamp

## Descrição do Projeto

Estudar no exterior pode ser emocionante e desafiador ao mesmo tempo. Mas o que pode estar contribuindo para isso? Uma universidade internacional japonesa decidiu investigar!

Use suas habilidades de manipulação de dados para explorar os dados de um estudo sobre a saúde mental de estudantes internacionais e descubra quais fatores podem ter o maior impacto.

## Conhecimentos reforçados:
- Big Six (SELECT, FROM, WHERE, GROUP BY, HAVING, ORDER BY)
- JOINS
- Agrupamento e Ordenação
