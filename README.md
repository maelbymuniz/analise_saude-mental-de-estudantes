# Análise da saúde mental de estudantes

Projeto 1 da trilha Analista de dados associado em SQL da plataforma de cursos DataCamp

## Descrição do Projeto

Estudar no exterior pode ser emocionante e desafiador ao mesmo tempo. Mas o que pode estar contribuindo para isso? Uma universidade internacional japonesa decidiu investigar!

Use suas habilidades de manipulação de dados para explorar os dados de um estudo sobre a saúde mental de estudantes internacionais e descubra quais fatores podem ter o maior impacto.


## Instruções do Projeto

Explore e analise os dados dos estudantes para verificar como o tempo de permanência (*stay*) impacta as médias das pontuações de diagnósticos de saúde mental dos estudantes internacionais presentes no estudo.

- Retorne uma tabela com nove linhas e cinco colunas.
- As cinco colunas devem ser nomeadas (usando alias) da seguinte forma:
*stay*, *count_int*, *average_phq*, *average_scs* e *average_as*, nessa ordem.
- As colunas de média devem conter a média das colunas todep (teste PHQ-9), tosc (teste SCS) e toas (teste ASISS) para cada tempo de permanência, arredondadas para duas casas decimais.

- A coluna *count_int* deve indicar o número de estudantes internacionais para cada tempo de permanência.

- Ordene os resultados pelo tempo de permanência (*stay*) em ordem decrescente.

**Observação:** Criar novas células no workbook renomeará o DataFrame. Certifique-se de que sua solução final use o nome df.

## Conhecimentos reforçados:
- Big Six (SELECT, FROM, WHERE, GROUP BY, HAVING, ORDER BY)
- Agrupamento e Ordenação
